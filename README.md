# Watog Ionic Mobile App 

## Dependencies

1. SQLite

- `ionic cordova plugin add cordova-sqlite-storage`
- `npm install --save @ionic-native/sqlite`

2. Native Storage

- `ionic cordova plugin add cordova-sqlite-storage`
- `npm install --save @ionic-native/sqlite`

3. Camera/Photo

- `ionic cordova plugin add cordova-plugin-camera`
- `npm install --save @ionic-native/camera`
